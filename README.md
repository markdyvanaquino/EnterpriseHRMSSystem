This module was built during my work on an enterprise HRMS system at Hi-Precision Diagnostics (Mar 2017 – Apr 2018).
In addition to the core HR features like employee management and payroll, I worked on a resource management component that handled internal assets and usage tracking. The goal was to support day-to-day operations by providing a simple but structured way to manage shared resources and user activity.
I implemented the backend using ASP.NET, designed the data models, and handled the database interactions with SQL Server through Entity Framework. The system follows a standard MVC structure, keeping the logic organized and maintainable.
